# Medical Apps Monorepo

This repository contains multiple medical-related Flutter applications.

## Projects

### 1. Google Map Clone
A Flutter application for finding nearby medical services with map integration.
- Location: `google_map_clone/`
- Features: Medical service search, Google Maps integration, Prefecture-based location selection

### 2. MediNavi
Medical navigation and healthcare services application.
- Location: `medinavi-mac/`
- Features: Healthcare mapping and navigation

## Getting Started

Each project can be run independently. Navigate to the respective project directory and follow the setup instructions.

### Running Google Map Clone
```bash
cd google_map_clone
flutter pub get
flutter run
```

### Running MediNavi
```bash
cd medinavi-mac
flutter pub get
flutter run
```

## Development

Each project maintains its own dependencies and configuration. Make sure to run `flutter pub get` in each project directory before running.
